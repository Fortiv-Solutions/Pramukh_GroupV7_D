# Pramukh_GroupV7_D

Official website for Pramukh Group — built with Next.js 16, React 19, Framer Motion, and GSAP.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Framework**: Next.js 16.2.9
- **UI**: React 19
- **Animations**: Framer Motion + GSAP
- **Smooth Scroll**: Lenis
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
